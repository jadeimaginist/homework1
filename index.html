  1. Introduction

    2. The visualization case comes from National Data published by National Bureau of Statistics of China. It's about national foreign currency reserve from 2011 to 2015.

  2. Why it's confusing

    3. The initial chart emphasised every single value using big circle area but delibrately played down the changes using small circle area.

  3. Justification

    4. Here I use the line chart to show how the national foreign reserve changes over these 5 years. If using the first way to visualize data, viewers can easily see how the values are big and overlook the changes, especially the decline in 2015.
